# libxsvf-xvc
A quick'n'dirty tool using libxsvf (https://clifford.at/libxsvf) to "play" (X)SVF files
to a XilinxVirtualCable (https://github.com/Xilinx/XilinxVirtualCable) server.
No command line parameter support so far - simply adjust main.c to your needs.

**WARNING**: This tool does no error handling whatsoever! You have been warned.
